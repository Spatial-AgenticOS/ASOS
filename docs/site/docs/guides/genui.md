# Gen-UI (SDUI)

This page is a short mirror of the Mintlify guide.

Canonical guide: `docs/mintlify/guides/genui.mdx`.

FERAL Gen-UI is server-driven UI:

- Brain emits structured UI payloads.
- Web client renders supported SDUI components.
- Patch updates (`sdui_patch`) update existing surfaces.

Current limits:

- Not arbitrary HTML generation.
- Not guaranteed native parity across every client.
- Marketplace trust/signing flows are still evolving.
